# ATNDIA — Sua loja atendendo antes do cliente esfriar

Assistente comercial por Inteligência Artificial (IA) especializado para lojas de carros, concessionárias e revendas multimarcas. A **ATNDIA** atende, qualifica e prepara o lead para o vendedor fechar o negócio com muito mais assertividade.

## 🚀 O que é a ATNDIA?

A ATNDIA é uma plataforma inovadora criada para resolver o maior gargalo das concessionárias de automóveis: **a demora e a falta de padrão no primeiro contato do lead**.

Ela não substitui a equipe comercial; ela atua como um braço de suporte inicial que qualifica o cliente 24 horas por dia, 7 dias por semana, enviando o lead "quente" e estruturado diretamente para o vendedor no WhatsApp.

## 📦 Estrutura do Projeto

O repositório contém os arquivos da Landing Page institucional de alta performance da marca:

* `index.html`: Estrutura HTML5 semântica e SEO-friendly contendo a cópia de vendas completa baseada no comportamento do lojista de automóveis.
* `style.css`: Estilização Vanilla CSS premium baseada na paleta oficial do **Design System Blue3** (Dark Mode com tons de preto asfalto, azul IA, verde oportunidade e glows dourados).
* `conversas_mockup.svg`: Mockup interativo em SVG de uma conversa real simulando o atendimento da IA qualificando um lead interessado em um veículo e agendando a visita.
* `.gitignore`: Configuração para ignorar arquivos locais indesejados.

## 🛠️ Tecnologias Utilizadas

* **Estrutura**: HTML5 Semântico
* **Estilização**: Vanilla CSS (CSS Moderno e Responsivo)
* **Tipografia**: Google Fonts (*Space Grotesk*, *Plus Jakarta Sans*, *Inter*)
* **Gráficos**: SVG Vetorial Nativo

## ⚡ Hospedagem Instantânea na Vercel

Este projeto está 100% otimizado para ser hospedado gratuitamente e em segundos na **Vercel**:

1. Crie uma conta ou faça login em [Vercel](https://vercel.com).
2. Clique em **Add New** ➔ **Project**.
3. Importe este repositório do GitHub.
4. A Vercel identificará automaticamente o projeto estático. Basta clicar em **Deploy**.
5. No painel da Vercel, conecte o seu domínio próprio **`atndia.com`** de forma simples e rápida!
